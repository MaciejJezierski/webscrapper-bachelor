Porównywarka cen książek - aplikacja bazodanowa obsługująca skrypty do web scraping’u

Technologia:
I Backend:
Java + Spring MariaDB

II Frontend:
HTML + CSS + JS + Bootstrap

III Web Scraping:
Python3 + beautifulsoup


Funkcjonalności:

Użytkownik niezalogowany:
Przeglądanie podstawowych zasobów strony (bazy danych aktualnych i archiwalnych cen książek)
“Wyszukiwanie wielu”

Użytkownik zalogowany:
Zachowanie wyników wyszukiwania (ulubione)
Analizowanie wyników (wykresy/widoki)
Zgłaszanie propozycji nowych sklepów oraz problemów/błędów strony

Admin:
Dodawanie skryptów dla nowych sklepów
Rozpatrywanie próśb użytkowników
